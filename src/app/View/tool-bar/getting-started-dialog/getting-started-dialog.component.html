Getting Started


  • Simulation

    The buttons are in order from left to right: Micro-Step, Run, Macro-Step, Load. The program has to be loaded everytime it changes, wether it is by manual changes, imports or loaded examples. The execution has three variations. Micro-Step executes a single line of the micro program, macro step is doing the same for the macro program. Run will execute the program until it terminates succesfully or runs into a fault state. The animation speed can be adjusted to showcase the execution of a program in the visualization but will impact execution speed heavily.


  • Import/Export

    Programs can be exported to and imported from line-separated text files (.txt). Make sure to reload the program after importing prior to execution.


  • Preferences

    For readability the presentation mode will scale up the font sizes. Do not use browser scaling as it will also scale buttons and other elements.


  • Getting Started

    The Toolbox comes with three preloaded programs in the Getting Started section. You can use them to get a first idea of the functionality of the MIC-1 or start altering them towards your own ideas.
